# Visual Studio Code - IDE Jupyter

## How To Install
### Install IDE Jupyter
download from release
### Install extensions
Please Download and install the following extensions manually
### add parameter to argv.json
Modifying a configuration file argv.json
windows
```
C:\Users\Administrator\.vside-jupyter\argv.json
```
MacOS

add
``
"enable-proposed-api":["ms-python.python","ms-toolsai.jupyter"]
```
Use developer tools to check for other errors
```
Developer: Toggle Developer Tools
```
